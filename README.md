# ✍️ Laravel Blog Project (Learning Track)

Проект персонального блога, созданный в процессе перехода с CMS-разработки (Bitrix/WP) на фреймворк **Laravel 9**. 
Здесь я документирую свой путь обучения, применяя архитектурные стандарты и лучшие практики PHP.

---

## 📈 Дневник разработки и освоенные технологии:

*(Здесь будут появляться пункты по мере прохождения курса)*

### 🛠 Шаг 1: Окружение и Базовая Авторизация
* **Инфраструктура:** Развернута локальная среда разработки с использованием **Docker**. Настроена и подключена база данных **MySQL**.
* **Frontend Scaffolding:** Интегрирован **Bootstrap** через пакет `laravel/ui` для быстрого прототипирования интерфейса.
* **Authentication:** Реализована базовая система регистрации и входа пользователей с помощью `php artisan ui:auth`.
* **Environment:** Настроена конфигурация `.env` для корректной работы приложения с контейнеризированной БД.
